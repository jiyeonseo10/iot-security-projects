# Smart Door Lock with Intrusion Detection

## Description
A security system built with Arduino Uno in Tinkercad.

## Features
- Password authentication
- LCD display
- Servo-controlled door lock
- PIR motion detection
- Alarm buzzer
- Status LEDs

## Components
- Arduino Uno
- 4x4 Keypad
- LCD 16x2 I2C
- Servo Motor
- PIR Sensor
- Buzzer
- Red/Green LED
